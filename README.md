# Bakalaurinis_Darbas

This repository contains the code and experiments for my bachelor thesis “Genetic Algorithms and Their Application in Natural and Exact Sciences”.

Thesis reviews genetic and other metaheuristic optimization algorithms, their operating principles, theoretical foundations and practical applications for solving optimization problems. The practical part focuses on several experiments implemented in Python, including memetic algorithm based motif search in DNA sequences, arithmetic expression optimization using genetic algorithms and solving 4x4 and 9x9 Sudoku puzzles using different metaheuristic approaches.

The performed experiments investigate how various metaparameters such as population size, mutation probability, swarm size and iteration count affect algorithm convergence speed, stability and solution quality. Different algorithms, including Genetic Algorithms (GA), Particle Swarm Optimization (PSO), Ant Colony Optimization (ACO) and Tabu Search are compared on 4x4 and 9x9 Sudoku problems.

The obtained results demonstrate that parameter selection has a significant impact on optimization performance and that different metaheuristic approaches exhibit different strengths depending on the complexity and structure of the problem.
