# Bakalaurinis_Darbas

This repository contains the code and experiments for my bachelor thesis.

## Files

- `MA_program.ipynb` – memetic algorithm for DNA motif search.
- `GA_program.ipynb` – genetic algorithm for arithmetic expression optimization.
- `GA_algorithm_Sudoku.ipynb` – genetic algorithm for 4x4 and 9x9 Sudoku.
- `ACO_Algorithm_Sudoku.ipynb` – ant colony optimization for Sudoku.
- `PSO_Algorithm_Sudoku.ipynb` – particle swarm optimization for Sudoku.
- `Tabu_Search_Sudoku.ipynb` – tabu search for Sudoku.

## Requirements

The notebooks were created and tested in Google Colab. They use Python and common libraries such as NumPy, pandas, tqdm and matplotlib.
